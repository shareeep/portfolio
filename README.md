# Shariff's Portfolio

Welcome to the repository for my portfolio website!
This site showcases my projects and journey as a software developer.

## Description & Features

- A fork of the Taxonomy Repo with Velite (for content management using .Mdx files)
- Built with Next.js
- Styled with Tailwind CSS
- Responsive Design
- Categorized project tag system with filtering capabilities

## Setup

Install dependencies:

```bash
pnpm i
```

Run the development server:

```bash
pnpm dev
```

Website will be running on [http://localhost:3000](http://localhost:3000).

## Deployment

- Custom domain on Cloudflare
- CI/CD via Vercel with 2 branches -> a development and production environment respectively
